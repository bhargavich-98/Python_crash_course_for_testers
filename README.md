# 🧠 Python Foundations for Testers – Project-Based Learning

## 📌 Overview

This repository documents my journey to mastering core Python concepts essential for automation testing. Instead of passive learning, this approach focuses on **building real projects** and applying the **Feynman Technique** to deeply understand each concept.

---

## 🎯 Learning Goals

* Understand core Python fundamentals
* Apply concepts through hands-on projects
* Explain concepts in simple terms (Feynman Technique)
* Build a strong base for automation testing (SDET path)

---

## 🧩 Topics Covered

* Variables & Data Types
* Conditional Statements (if/else)
* Loops (for, while)
* Functions & Scope
* String Manipulation
* f-Strings

---

## 🛠 Projects


### 1. Login System

A login system that validates password rules and allows limited attempts before locking the user.

### 2. Calculator System

**Concepts:** Functions, scope, reusable code

A menu-driven calculator that performs basic arithmetic operations using functions.

---

### 3. Password Analyzer

**Concepts:** String manipulation

**Description:**
Analyzes password strength based on rules like length, uppercase letters, digits, and special characters.

---

### 4. Test Report Generator

Simulates test execution and generates formatted test reports.

## 🧠 Learning Approach (Feynman Technique)

For each concept:

1. Learn the concept
2. Build a project
3. Explain it in simple words
4. Identify gaps and relearn

---

## 🔁 Practice Strategy

* Write code daily
* Break and debug your own code
* Avoid copying solutions
* Focus on understanding, not memorization
